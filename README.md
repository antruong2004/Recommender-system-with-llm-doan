# Recommender-system-with-llm-doan

# 🛒 ĐỀ TÀI: ỨNG DỤNG MÔ HÌNH NGÔN NGỮ LỚN (LLM)

## Giới thiệu
Dự án xây dựng hệ tư vấn thương mại điện tử kết hợp LLM, RAG, vector search và recommender.

## Công nghệ
- Python, Flask
- Groq API
- PostgreSQL / JSON fallback
- Sentence Transformers
- Hybrid recommender

## Chức năng chính
- Chatbot tư vấn sản phẩm
- Tìm kiếm ngữ nghĩa
- Gợi ý sản phẩm cá nhân hóa
- Phân tích dữ liệu TMĐT

## Chạy dự án
```bash
pip install -r requirements.txt
python app.py
```

## Ghi chú
Dự án sử dụng dữ liệu mô phỏng và dữ liệu TMĐT để phục vụ nghiên cứu, demo và đánh giá hệ thống tư vấn.
