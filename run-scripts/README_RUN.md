# Run scripts for TechStore

## Files
- `run-menu.bat`: double-click to open the run menu
- `run-menu.ps1`: PowerShell menu with all modes
- `run-lite.ps1`: fastest mode, no vector/recommender
- `run-ai.ps1`: lightweight chat mode
- `run-semantic.ps1`: vector search enabled
- `run-full.ps1`: full system
- `run-recommender.ps1`: recommender only

## Recommended usage
- UI work: `run-lite.ps1`
- Chat testing: `run-ai.ps1`
- Semantic search testing: `run-semantic.ps1`
- Demo: `run-full.ps1`
